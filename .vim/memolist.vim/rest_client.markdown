RESTClient
===

## Cookie

* 開発ツール > Cookie > Cookie情報の表示
* RESTClient > Headers > Custom Header
```
Name:  Set-Cookie
Value: id=1
```

## POST

* RESTClient > Headers > Custom Header
```
Name:  Content-Type
Value: application/x-www-form-urlencoded
```
* RESTClient > Body
```
id=1&name=KazuakiM
```
